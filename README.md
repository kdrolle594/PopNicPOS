
  # All-in-One Restaurant POS System (Vue + JavaScript)

  This project has been recreated as a Vue 3 + JavaScript application powered by Vite.

  ## Tech Stack

  - Vue 3 (Composition API)
  - JavaScript (no TypeScript required for app code)
  - Vite
  - Tailwind CSS
  - localStorage persistence for POS data

  ## Available Modules

  - Dashboard
  - POS Terminal
  - Kitchen Display
  - Inventory Management
  - Menu Management
  - Loyalty Program
  - Analytics & Reports

  ## Run Locally

  1. Install dependencies:

    `npm install`

  2. Start the development server:

    `npm run dev`

  3. Build for production:

    `npm run build`
